Commentary — how far can the complaint ratio fall?

1 · How much an intervention buys. The upgrade cut the non-outage complaint ratio ~44% — from 46.4% (pre) to 25.8% (immediately post) — and almost all of it landed in the first two months. From there it plateaued (→ 24.4% now). A single intervention delivers a big one-time step-change, then flattens.

2 · The idealistic floor. It never reaches zero — a live network always throws off some genuine complaints — but it can fall far below today's 24%. The strongest CSP here runs at ~6% (K.H.), and several sit in the low-to-mid teens. The direction to aim for is single digits; 24% still carries heavy, fixable excess. (Absolute target to be set by ops, not by the current best.)

3 · What it holds. The plateau + earlier mean-reversion signal say the residual gap closes only with recurring maintenance (the ratio drifts back up ~2–3 months after a fix), not one-off upgrades. Biggest lever = the laggards — Gaurav 43%, MD 36%, Nextia 31% run 2–3× the best CSPs; pulling them down moves the average most. Runway so far: 46% → 24% (−44%, banked); the next leg toward single digits is a maintenance-discipline problem, not a capital one.
